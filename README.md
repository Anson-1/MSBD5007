# Final Project - XU Shiyang

Brief README for the code in this workspace.

## Overview
This repository contains implementations and experiments for the final project by XU Shiyang. It includes algorithm implementations, data utilities, and verification scripts used to run and evaluate the methods.

## Files
- `main.py` — Project entry point; runs experiments or the pipeline.
- `data.py` — Data loading and preprocessing helpers.
- `admm.py` — Implementation of the ADMM algorithm used in experiments.
- `bcd.py` — Block Coordinate Descent implementation.
- `proposed.py` — Implementation of the proposed method from the project.
- `verify.py` — Scripts for verifying results and computing evaluation metrics.
